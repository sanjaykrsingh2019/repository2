How to Set Up Sample Data

Local Run Setup Instructions
To set up your data to run locally, please copy the Samples folder from the project's directory to the local run DataRoot folder. The DataRoot folder is defined in the Data Lake options at
  • Tools → DataLake → Options and Settings ... → Azure Data Lake → General → U-SQL Local Run → DataRoot
Cluster Run Setup Instructions
To set up your data to run on the cluster, please copy the Samples folder from the project's directory to the root of the dependent Data Lake Store
